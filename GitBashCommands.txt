git clone URL - klonira repository-to
git status - pokazva statusa na lokalnoto repository
git add . - selektira failovete koito iskame da dobavim v lokalnoto repository
git commit -m "Text massage about the current change in the commit" - komitva (izprashta) novite promeni kym lokalnoto repository
git push - vkarva promenenite failove koito sme izbrali v centralnoto repository(git server)
